# Label Definition Review: wornow2023_ehrshot

## Review Status

Complete.

## Label Summary

EHRSHOT defines 15 benchmark labels across operational outcomes, lab test results, new diagnoses, and chest X-ray findings.

## Label Type

Structured EHR-derived benchmark labels, with one chest X-ray findings task derived from radiology-report labels generated by CheXpert.

## Target Construct

General clinical prediction tasks, not dementia or cognitive impairment.

## Timing

The benchmark defines prediction times and horizons for each task. Operational outcomes have task-specific windows; lab, new-diagnosis, and chest-X-ray tasks use uniform prediction-window designs within category.

## Strengths

The paper provides explicit task categories, train/validation/test splits, and few-shot sampling definitions. This is useful for our review's emphasis on clear index time and horizon definitions.

## Weaknesses

Labels are not clinically adjudicated. Some low-prevalence tasks have very few positive cases, which limits estimate precision. The selected cohort is enriched to support task evaluation and is not a general sample.

## Risk of Label Leakage

The authors use prediction times and prior timelines for modeling, which is a strength. Leakage risk would need task-specific audit; the review did not inspect code or labels.

## Use in Review

Use as a benchmark-design example, not as evidence about cognitive-risk label construction.
