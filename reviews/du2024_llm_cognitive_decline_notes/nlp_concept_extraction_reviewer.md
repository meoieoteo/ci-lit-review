# NLP / Concept Extraction Review: du2024_llm_cognitive_decline_notes

## Review Status

Complete.

## NLP Task

LLM and model-based classification of note sections for cognitive decline, with keyword/rationale extraction.

## NLP Methods

GPT-4, Llama 2, hard prompts, RAG, few-shot prompting, error-analysis instructions, attention-based DNN, XGBoost, and majority-vote ensemble.

## Concept Handling

GPT-4 identified broader phrase-level concepts, medication mentions, dementia subtypes, and emotional/behavioral terms beyond curated keywords.

## Strengths

The paper compares general-domain LLMs to strong local models and analyzes distinct error profiles.

## Limitations

LLM outputs are prompt-sensitive and model-version-sensitive. Extracted keywords are interpretability aids, not normalized clinical concepts.

## Use in Review

Use as evidence that LLM concept interpretation can complement traditional NLP, but needs task-specific validation.
