# EHR Schema / Common Data Model Review: gilmorebykovskyi2018_ci_ehr_phenotype

## Review Status

Complete.

## Data Model

Medicare claims linked to local hospital EHR free-text documentation. No common data model is described.

## Structured Data Elements

Medicare ICD-9 claims identify dementia and index hospitalization diagnoses.

## Unstructured Data Elements

Free text from flowsheets and clinician notes across physicians, nurses, social workers, physical therapists, and occupational therapists.

## Temporal Structure

The reviewed text window is the last three hospital days.

## Portability Concerns

Terminology may vary across disciplines, hospitals, EHRs, and care settings.

## Use in Review

Use to motivate inclusion of flowsheets and multidisciplinary notes, not just physician progress notes.
