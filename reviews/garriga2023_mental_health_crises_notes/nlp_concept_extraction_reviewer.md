# Specialist Review: NLP Concept Extraction

## Specialist Scope

BERT note representations for mental-health crisis prediction.

## Bottom Line

Useful as evidence that note embeddings add signal, but not a concept-extraction paper for cognitive impairment.

## NLP Task and Clinical Concept Summary

Document/semantic representation for risk prediction.

## Note Sources and Text Window

Mental-health clinical notes before weekly prediction points.

## Concept Inventory Relevance

Low for dementia concept inventory.

## Annotation and Reference Standard

No note-level concept annotation is central.

## Extraction Method

BERT-derived 768-dimensional note embeddings.

## Context Handling

Opaque embedding approach; no interpretable assertion handling.

## Temporality and Leakage

Rolling prediction design requires note cutoff before weekly index.

## Evaluation Design

Model-level evaluation using AUPRC/AUROC.

## Error Analysis

SHAP interpretation is used for model interpretation.

## Portability and Implementation

Embedding utility depends on local notes and clinical domain.

## Fit to Pre-Cardiology Risk Scoring

Method analogy only.

## Strengths

Demonstrates additive value of notes in a low-prevalence task.

## Concerns

Opaque embeddings and non-dementia target.

## Missing Information

Concept-level errors and note-type details.

## Implications for This Literature Review

Use as support for testing note embeddings as a comparator to concept features.

## Recommended Follow-up

Do not add concepts from this paper to dementia inventory.
